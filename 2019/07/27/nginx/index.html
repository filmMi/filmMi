    一键安装四个依赖

    yum -y install gcc zlib zlib-devel pcre-devel openssl openssl-devel

    创建一个文件夹

    cd /usr/local
    mkdir nginx
    cd nginx

    安装nginx包

    wget http://nginx.org/download/nginx-1.13.7.tar.gz

    如果提示失败，安装wget后，再安装nginx

    yum -y install wget

    解压包

    tar -xvf nginx-1.13.7.tar.gz

    进入nginx目录

    cd /usr/local/nginx/nginx-1.13.7/

    执行命令

    ./configure

    执行make命令

    make

    执行make install命令

    make install

    启动

    /usr/local/nginx/sbin/nginx -s reload

    查看nginx进程是否启动：

    ps -ef | grep nginx

    或者打开游览器输入ip地址，显示成功。
    停止服务

    /usr/local/nginx/sbin/nginx -s stop

    配置文件加载后重新启动

    /usr/local/nginx/sbin/nginx -s reload

    打开配置文件

    vi /usr/local/nginx/conf/nginx.conf
    i (修改操作)
    先esc然后:wq保存退出

    修改了配置文件需要重新启动才能生效

    服务器多项目nginx配置文件修改

    打开配置文件
    vi /usr/local/nginx/conf/nginx.conf
    只修改server部分
    //第一个项目打包文件
    location /pro {
    alias /home/shianyuan/pro;
    index index.html index.htm;
    }
    //第二个项目打包文件
    location /blog{
    alias /home/shianyuan/blog;
    index index.html index.htm;
    }

    重启后运行。

    注意点：如何重启失败，可能是配置文件报错。操作前注意备份nginx.conf文件.
